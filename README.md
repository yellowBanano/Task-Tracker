# Task-Tracker

Technologies:
* Java 8
* Spring
* Hibernate
* MySQL
* React.js
* Thymeleaf

To start application:
1. Install Node.js
2. In node console cd to application folder
3. Write "npm install"
4. Write "npm run-script watch"
5. Run SQL script
6. Build application with Apache Tomcat or any analog
7. Go to localhost
8. Profit!
